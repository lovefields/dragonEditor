# dragonEditor

## Support browser
- Chrome 53+
- Safari 13+
- Firefox 63+
- Edge 18+ (44+)
- IOS 11+
- Android 7+

## Using Plugin
- `highlight.js`

## Using
```js
const editor = dragonEditor.init(".editor-dragon");
```
