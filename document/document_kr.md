# Dragoneditor 문서

## 사용법
```js
const editor = new dragonEditor(wrap, {
   'options' : 'value'
});
```

#### `wrap`
- 값 : CSS Selector
- 기본값 : `.editor_area`
- `null`을 넣을 경우 기본값으로 작동합니다.

#### 필수 옵션
- [mediaUploadURL](#mediaUploadURL)
- [mediaUpdateURL](#mediaUpdateURL)
- [mediaDelURL](#mediaDelURL)
